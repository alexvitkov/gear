sova 'documentation'

Variables

Initial declaration is done with :=, reassignment with =

:= will declare the variable in the current scope, shadowing variables with the same name in parent scopes.

= will not declare a new variable - if the variable doesn't exist in the current scope it will recursively look for it in parent scopes. 

          > foo := 100 + 50
          
          > foo
          150
          
          > foo = foo + 30
          150

Booleans

The boolean constants are called true and false.

Numbers

The only numeric type (right now) is 64 bit float. +,-,*,/ >, <, >=, <= work as you would expect.

Dictionaries

          > user := dict();
          > user.first_name = "John";
          > user.last_name = "Doe";

          > user
          dict {
              first_name: "John",
              last_name: "Doe",
          }

          > user.first_name + " " + user.last_name
          John Doe

Quote/Eval

You can surpress evaluation of an expression with a single quote. The AST object will be returned so you can hack on it and evaluate it later.
